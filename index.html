Principles

How we build

Every tool we ship follows the same three constraints.

01
Speed by default

Rust isn't a style choice — it's a contract. Our tools are fast enough to disappear into your workflow.

02
Zero friction setup

If it takes more than one command to install, we haven't finished building it yet.

03
AI-first, not AI-only

Tools that augment AI agents without locking you out. Everything works standalone too.

04
Open by nature

We build in the open because that's how the best tools get built — with the community, not at them.
